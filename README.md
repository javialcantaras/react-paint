# PAINT
## Technologies
  - React
  - Redux
  - React canvas draw component
  - Mocha
  - Chai

## Getting started
  - Node: v9.8.0

### Installation
```
$ git clone https://javivi91@bitbucket.org/javivi91/atrapalo.git
$ cd atrapalo
$ npm install
```
### Run
```
$ npm start
```
### Test
```
$ npm test
```
